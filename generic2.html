Partes de un elemento:

Vínculos: Parte más importante de una página web, creación de conexión con otras paginas web, para implementar un vinculo usamos la etiqueta

Imágenes: Incrusta una imagen en la página, en la posición en que aparece, lo logra a través del atributo SRC (SOURCE) el cual contiene el PATH (ruta o ubicación) de tu archivo de imagen.

Elementos vacíos: no tienen etiqueta de cierre,ni contenido cerrado. No es un lenguaje de programación, es un lenguaje de marcado y de texto compuesta por etiquetas que se emplean para crear la estructura completa de una página web siendo la base de estas.
