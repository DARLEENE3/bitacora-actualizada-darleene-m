ARDUINIO CONFIGURACIONES DE PINES

Tenemos 3 principales sentencias de configuraciones:

pinMode (pin,mode): Usado para asignar el modo de un pin, puede ser al número que se quiere asignar 0-19. Puede ser INPUT o OUTPUT.

digitaWrite (pin,value): Una vez un pin es asignado como OUTPUT se puede asognar a HIGH (5 volts) o LOW (0 volts). Básicamente siginifica ON y OFF.

int digitialRead (pin): Una vez un pin es asignado como INPUT se puede usar para devolver HIGH (5 volts) o LOW (0 volts).
